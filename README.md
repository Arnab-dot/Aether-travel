## üåü Overview

**BondhuChol** (Bengali: "Let's Go Together") is an intelligent group travel planning platform that uses Natural Language Processing (NLP) and sentiment analysis to help groups of friends decide on travel destinations. The system analyzes individual preferences and descriptions to recommend destinations that maximize group satisfaction.

### üéØ Problem Statement

We've all been there‚Äîsitting with friends, excited to plan a trip, only to end up stuck in endless debates because everyone wants something different. One person dreams of mountains, another insists on beaches, and someone else wants a city escape. Instead of excitement, confusion takes over and the plan often fades away.

### üí° Solution

BondhuChol provides an AI-powered decision-making system that:
- Collects individual travel preferences from all group members
- Analyzes sentiment and confidence scores using NLP
- Recommends destinations that best match the group's collective preferences
- Eliminates conflicts and makes group travel planning stress-free

---

## ‚ú® Features

### Core Features

- **üîê User Authentication**
  - Secure JWT-based authentication
  - User registration and login
  - Session management with token refresh

- **üë• Group Management**
  - Create travel groups with unique IDs
  - Join existing groups
  - View group members and their preferences

- **üåç Destination Preferences**
  - Add unlimited custom destinations (cities, countries, landmarks)
  - Detailed descriptions for each destination
  - Multi-destination support per user

- **ü§ñ AI-Powered Recommendations**
  - Sentiment analysis using TextBlob
  - Confidence score calculation
  - Group consensus algorithm
  - Individual preference tracking

- **üìä Analytics Dashboard**
  - Group confidence metrics
  - Individual sentiment scores
  - Destination preference visualization

### Technical Features

- RESTful API architecture
- Real-time data processing
- Responsive React frontend
- Scalable Django backend
- PostgreSQL/SQLite database support
